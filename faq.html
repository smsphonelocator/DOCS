FAQ – SMS Phone Locator

Dernière mise à jour : 2025

Voici les réponses aux questions les plus fréquentes concernant l'application SMS Phone Locator : localisation par SMS, commande RING, double SIM, précision GPS, etc.

1. La commande GPS ne fonctionne pas, que faire ?

Certains téléphones (Google Pixel, Samsung récents, Xiaomi…) filtrent les SMS très courts comme « GPS code ».

Solution : envoyer une phrase complète, par exemple :

« Peux-tu te localiser avec GPS code ? »

L’application détecte la commande où qu’elle soit dans le message.
